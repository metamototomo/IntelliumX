## Adding a Tag

Right-click any document and choose **Add Tag**, then type a tag name and press **Enter**.

## Tag View

Click the **Tags** tab in the File Manager to browse all documents grouped by tag. Click any tag to filter the list.

## Pinning a Tag

Click the pin icon next to a tag name to keep it in your quick-access bar.
