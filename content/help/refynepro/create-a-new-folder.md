## How to Create a New Folder

##### 1. Navigate to the desired folder and click the **New Folder** button.

![](2026-03-30-23-34-08.png)

##### 2. Enter a name for your file and press **Enter**.

![](2026-03-30-23-33-23.png)

##### 3. Once created, the folder will appear in both the File Manager and the Folder View, as shown below.

![](2026-03-30-23-39-38.png)
