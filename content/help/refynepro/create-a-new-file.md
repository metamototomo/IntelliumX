## How to Create a New File

##### 1. Navigate to the desired folder and click the **New Document** button.

![](2026-03-29-23-19-43.png)

##### 2. Enter a name for your file and press **Enter**.

![](2026-03-29-23-09-38.png)

##### 3. Once created, the file will open automatically for editing.

![](2026-03-29-23-20-44.png)
