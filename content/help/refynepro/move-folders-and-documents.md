## Move a Document or Folder

1. Right-click the item you want to move, or select it and press **Cmd+K**.
2. Choose **Move** from the context menu.
3. Select the destination folder and confirm.

## Drag and Drop

You can also drag any document or folder directly onto a target folder in the File Manager panel.
