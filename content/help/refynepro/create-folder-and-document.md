## Create a Folder

1. In the **File Manager** panel, click the **New Folder** button.
2. Enter a name and press **Enter** to confirm.

## Create a Document

1. Select the folder where you want the document.
2. Click the **New Document** button, or press **Cmd+K** and choose **New Document**.
3. Enter a title and press **Enter**.

The document opens automatically in the editor ready for input.
