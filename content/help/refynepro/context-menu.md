## Opening the Context Menu

Right-click on any document or folder, or press **Cmd+K** anywhere in the editor, to open the context menu.

## Available Actions

The context menu adapts based on what you have selected:

- **Document selected** — Rename, Move, Duplicate, Delete
- **Folder selected** — Rename, Move, Delete
- **Text selected in editor** — Run AI Action, Copy, Cut

## Keyboard Shortcut

| Action | Shortcut |
|---|---|
| Open context menu | `Cmd+K` |
| Close context menu | `Esc` |
