## Global Shortcuts

| Shortcut | Action |
|---|---|
| `Cmd + K` | Open command palette |
| `Cmd + ,` | Open settings |
| `Cmd + /` | Toggle sidebar |

## Editor Shortcuts

| Shortcut | Action |
|---|---|
| `Cmd + Enter` | Submit prompt |
| `Cmd + Z` | Undo last action |
| `Cmd + Shift + C` | Copy response |
